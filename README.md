# CardSaathi

“Right Card. Right Benefit. Right Time.”

CardSaathi is an AI-powered financial assistant built for Bharat that recommends the best credit/debit card in real time to maximize cashback, rewards, and benefits.

## Problem
Most Indians miss cashback and rewards because they don’t know which card to use at the right time.

## Solution
CardSaathi uses Machine Learning and AI ranking algorithms to:
- Detect checkout moment
- Analyze merchant category
- Compare card rewards
- Recommend the best card instantly

## Technologies Used
- React / Flutter (Frontend)
- Python (Backend & AI Engine)
- Machine Learning Models
- AWS Cloud Infrastructure
- Secure REST APIs

## Key Innovation
- Real-time AI recommendation engine
- User profiling using ML
- Reinforcement learning improvement
- Built for Indian ecosystem (RuPay, Indian Banks)
